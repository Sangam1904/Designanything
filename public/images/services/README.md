# Service Background Images

This directory contains background images for each service page.

## Required Images:

1. **product-design-hero.jpg** - Background for Product Design & 3D Modeling service page
   - Recommended size: 1920x1080px
   - Should show 3D modeling, CAD work, or product design elements
   - Dark theme preferred for text readability

2. **technical-drawing-hero.jpg** - Background for Technical Drawing + GD&T service page
   - Recommended size: 1920x1080px
   - Should show technical drawings, blueprints, or engineering documentation
   - Dark theme preferred for text readability

3. **simulation-hero.jpg** - Background for Simulation & Analysis service page
   - Recommended size: 1920x1080px
   - Should show simulation results, FEA analysis, or engineering analysis
   - Dark theme preferred for text readability

4. **animation-hero.jpg** - Background for Animation & Rendering service page
   - Recommended size: 1920x1080px
   - Should show rendered products, animations, or visual presentations
   - Dark theme preferred for text readability

## Image Guidelines:
- Use high-quality, professional images
- Ensure good contrast for white text overlay
- Images should be relevant to each service
- Consider using dark backgrounds or adding dark overlay for better text readability
- Optimize file sizes for web (under 500KB recommended)
